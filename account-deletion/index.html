Account & Data Deletion

How to delete your data from TerminaLLM.

TerminaLLM does not require an account. All data is stored locally on your device. There is no server-side account to delete.

Delete All Data

Uninstall the App

Uninstalling TerminaLLM from your device permanently deletes all locally stored data, including:

  • SSH connection profiles
  • Credential Vault entries (passwords, private keys)
  • TOTP MFA secrets
  • Host key fingerprints
  • Security audit logs
  • AI chatbot conversation history
  • Terminal session recordings
  • Custom terminal themes
  • All app preferences and settings

This action is irreversible. We recommend creating an encrypted backup before uninstalling if you want to preserve your settings.

Delete Specific Data

You can selectively delete data without uninstalling:

Connection Profiles

Open the app → Connection screen → Swipe left on a profile → Delete. This removes the saved host, port, username, and any associated Credential Vault entry.

Credential Vault

Open the app → Settings → Security → Credential Vault → Delete individual entries or clear the entire vault.

Audit Logs

Open the app → Settings → Security → Audit Log → Clear All. You can also export logs as JSON before clearing.

AI Chatbot History

Open the app → Chatbot → Clear Conversation. This removes all stored messages from encrypted storage.

Terminal Recordings

Open the app → Settings → Recordings → Delete individual recordings or all recordings.

Custom Themes

Open the app → Settings → Terminal → Themes → Delete individual custom themes.

Server-Side Data

Proxy server data: If you used the built-in AI chatbot in free or pro mode, your device identifier and daily message count were sent to our proxy server for rate limiting. This data is ephemeral and not permanently stored. To request deletion of any residual proxy data, email privacy@terminallm.app (see below).
