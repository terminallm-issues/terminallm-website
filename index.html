Edit Code Remotely

Built-in code editor with syntax highlighting. Edit files on your server, save instantly, and see changes reflected in your AI agent's context. No laptop required.

Preview Your App

Port forwarding with an in-app browser. Preview localhost dev servers (React, Vite, Django, Rails) directly inside TerminaLLM. Edit, refresh, iterate, all from your phone.

Watch AI Changes Live

See every file your AI agent creates, modifies, or deletes in real-time. Line-by-line diffs, file counts, and export. Know exactly what happened while you were away.

Any AI Agent

Claude Code, Aider, Cursor, GitHub Copilot, OpenAI Codex, Google Gemini, Amazon Q, Amp, OpenCode. If it runs in a terminal, it runs in TerminaLLM. Not locked to one vendor.

Smart Push Notifications

Get notified when your AI agent needs input, not just when your terminal is idle. Process tree detection knows when an AI tool is actually waiting. Tap the notification to auto-reconnect.

Persistent Sessions

Your AI agent keeps running when you close the app. dtach-powered session persistence with scrollback replay. Reconnect and pick up right where you left off.

Screenshot a Bug

Take a photo of a bug, upload it to your server, and tell your AI agent "fix this." The file path is auto-inserted into the terminal. Visual debugging from your pocket.

Security First

AI Assistant Chat Pro

Voice to Shell Command Pro
