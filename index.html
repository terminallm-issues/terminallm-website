TerminaLLM

A remote workspace for command-line AI, from your phone. Edit code, preview your app, and watch every change, on sessions that survive every disconnect.

$ ssh you@your-server_

Edit Code Remotely

Built-in code editor with syntax highlighting. Edit files on your server, save instantly, and see changes reflected in your AI agent's context. No laptop required.

Preview Your App

Port forwarding with an in-app browser. Preview localhost dev servers (React, Vite, Django, Rails) directly inside TerminaLLM. Edit, refresh, iterate, all from your phone.

Watch AI Changes Live

See every file your AI agent creates, modifies, or deletes in real-time. Line-by-line diffs, file counts, and export. Know exactly what happened while you were away.

Any AI Agent

Claude Code, Aider, Cursor, GitHub Copilot, OpenAI Codex, Google Gemini, Amazon Q, Amp, OpenCode. If it runs in a terminal, it runs in TerminaLLM. Not locked to one vendor.

Smart Push Notifications

Get notified when your AI agent needs input, not just when your terminal is idle. Process tree detection knows when an AI tool is actually waiting. Tap the notification to auto-reconnect.

Persistent Sessions

Your AI agent keeps running when you close the app. Termina-TCP (an optional ~6.5 MB helper daemon) gives you near-instant reconnects over a long-lived TCP link; falls back to plain SSH if you'd rather not deploy it. dtach scrollback replay either way.

Live Activities on iOS

Watch SSH session status from the Lock Screen and Dynamic Island. Server name, live-ticking timer, and connection state always visible. Requires iPhone 14 Pro or later for Dynamic Island; Lock Screen works on any iOS 16.1+ device.

Screenshot a Bug

Take a photo of a bug, upload it to your server, and tell your AI agent "fix this." The file path is auto-inserted into the terminal. Visual debugging from your pocket.

Security First

Optional TOTP and biometric auth, AES-256-GCM encrypted credential vault, host key verification, multi-hop SSH through jump hosts, encrypted backups, audit logging, and screen capture prevention. Your keys stay on your device.
Three security profiles tune the trade-offs: Paranoid and Casual block screenshots; Speedster permits them for usability.

Command Palette

50+ commands organized by category, fuzzy-searchable from anywhere in the terminal. Jump to settings, share a session, paste a snippet, change themes — without leaving the keyboard.

Snippets with Variables

Save reusable command templates with prompts for the parts that change (server names, paths, ports). One tap fills in the variables and pastes a ready-to-run command. Sync across devices via encrypted backup.

15 Themes + Custom Editor

15 hand-tuned terminal color schemes plus a visual editor for your own. Import from iTerm2 .itermcolors or paste any 23-color JSON. Pick a different theme per connection if you want.

Split-Keyboard Landscape

Rotate to landscape and the keyboard splits into left/right halves around the terminal — thumb-friendly typing at full speed, with sticky Ctrl / Alt / Shift and arrow keys exactly where you'd put them on a real keyboard.

AI Assistant Chat Pro

Ask anything about your terminal, your code, or how to use TerminaLLM. Powered by Anthropic's Claude. Conversation history persists across sessions.

Voice to Shell Command Pro

Tap the wand button. Say what you want. Get the shell command. Natural language in, valid syntax out — no docs to look up, no flags to memorize.
