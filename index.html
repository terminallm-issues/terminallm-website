Swarm: a team of AI agents, run from your phone

Launch a crew of coding agents on your own server and watch it work. A lead architect breaks the job into pieces and hands them out, while the agents you choose run side by side, each in its own live terminal pane.

  • Run any agent: one-tap presets for Claude Code, Codex, and Aider, or point a pane at your own command, each with its model and effort.
  • Start from a recipe: Debugging, Code review, Add tests, Refactor, Implement a feature, or Write docs.
  • Each pane is a real terminal, not a log tail. Read it, type into it, take over any time.
  • Agents run on your machine, signed in with your own AI CLI logins. Your prompts and your code never pass through us.

Swarm needs a small helper daemon on your server, which the app can install for you (one tap on supported servers). You choose when it installs, and removal is one command.
