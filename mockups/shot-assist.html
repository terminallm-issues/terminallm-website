My node server dies when I close the app. How do I keep it running?
2:47 PM
TerminaLLM Assistant

Run it inside a persistent session so it is not tied to your connection. TerminaLLM sessions survive disconnects, so the server keeps running even after you close the app.

bashCopy
$ dtach -A ~/dev.sock -b 256k \
    npm run dev

Reconnect any time and you will drop straight back into the live output.
