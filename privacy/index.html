Privacy Policy

Last Updated: February 22, 2026

Overview

TerminaLLM ("the App") is a mobile SSH terminal client that enables secure remote command execution. This Privacy Policy describes how we collect, use, and protect your information.

Information We Collect

Data Stored Locally on Your Device

The App stores the following information locally on your device using platform-native encrypted storage (iOS Keychain / Android EncryptedSharedPreferences):

  • SSH Connection Details: Host address, port number, and username
  • TOTP MFA Secrets: Encrypted secrets for multi-factor authentication
  • Host Key Fingerprints: SSH server fingerprints for man-in-the-middle attack prevention
  • Security Audit Logs: Connection attempts, authentication events, and security-related activities (up to 1,000 most recent events)
  • Session Information: dtach session identifiers for session persistence
  • App Preferences: Biometric authentication settings, security profiles, terminal preferences
  • Credential Vault (Opt-in): If you enable the Credential Vault, SSH passwords, private keys, and passphrases are encrypted and stored locally using HKDF-Expand key derivation with per-credential nonces, XOR stream encryption, and HMAC-SHA256 authentication tags. Vault entries have configurable TTL expiry and are stored exclusively in platform-native encrypted storage
  • Media Drop (Camera/Photo Library): When you use the Media Drop feature, selected images or files are uploaded directly to your remote server via SFTP. Media files are not cached or stored locally by the App
  • Port Forwarding Rules: Saved port forwarding rules are stored locally per connection profile. No network traffic is intercepted or logged
  • AI Chatbot Usage: Daily message count for free-tier enforcement is stored locally. The last 50 chatbot messages are persisted in platform-native encrypted storage for conversation continuity
  • AI Chatbot API Key (BYOK): If you bring your own Anthropic API key, it is stored locally in platform-native encrypted storage. It is never transmitted to TerminaLLM servers
  • Terminal Session Recordings: If you use the recording feature, terminal sessions are captured in asciicast v2 format and stored locally. Recordings are not encrypted at rest and are never automatically uploaded
  • Encrypted Backups: Backup files are encrypted with AES-256-GCM using a password-derived key (PBKDF2). They are never uploaded to TerminaLLM servers
  • Custom Terminal Themes: User-created terminal color themes are stored in platform-native encrypted storage
  • Connection History: Timestamps, durations, and host information for recent connections (up to 10 entries) stored locally
  • Proxy Authentication Token: A JWT token containing only your device identifier and subscription tier
  • Device Identifier: A randomly generated identifier (not linked to your hardware or advertising ID)
  • iOS Live Activities: When an SSH session is active on iOS 16.1+, session metadata is displayed on the Lock Screen and Dynamic Island via iOS Live Activities

Data Transmitted to External Services

  • AI Chatbot (Free/Pro tiers): Messages are sent through the TerminaLLM proxy server (hosted on Fly.io) to the Anthropic API. Messages are not stored on the proxy server. See Anthropic's Privacy Policy
  • AI Chatbot (BYOK mode): Messages are sent directly from your device to the Anthropic API. No data passes through the TerminaLLM proxy

Data We Do NOT Collect

  • Passwords (by default): SSH passwords are not stored by default
  • Command History: Commands executed on remote servers are not logged by the App
  • Terminal Output: Content displayed in the terminal is not stored or transmitted
  • Location Data: We do not access your location
  • Personal Information: We do not collect names, emails, or other personal data

Voice Input

The App offers optional speech-to-text input for dictating terminal commands and chatbot messages. Voice recognition is performed entirely on-device using platform-native engines. No audio data is recorded, stored, or transmitted by the App.

How We Use Information

All data stored by the App is used solely to provide its functionality:

  • Connection Details: Enable quick reconnection to previously accessed servers
  • Credential Vault: Enable seamless reconnection without re-entering credentials (opt-in only)
  • TOTP Secrets: Verify your identity through multi-factor authentication
  • Host Key Fingerprints: Protect against man-in-the-middle attacks
  • Audit Logs: Allow you to review security events on your device
  • Session Information: Enable resumption of interrupted sessions
  • Device Identifier & Usage Count: Enforce free-tier daily message limits

Data Security

Encryption at Rest

  • iOS: All sensitive data stored in Apple Keychain with first_unlock_this_device accessibility
  • Android: EncryptedSharedPreferences with AES-256 encryption

Encryption in Transit

  • All SSH communications use industry-standard SSH protocol encryption
  • Host key verification prevents man-in-the-middle attacks
  • AI chatbot communications use HTTPS/TLS

Additional Security Measures

  • Multi-factor authentication (TOTP) required for app access
  • Optional biometric authentication (Face ID / Fingerprint)
  • Session timeouts after inactivity
  • Rate limiting on authentication attempts
  • Screen capture prevention (Android)
  • Clipboard auto-clear after paste operations

Data Retention

  • Connection Details: Retained until you uninstall the App or clear app data
  • Audit Logs: Last 1,000 events retained, older events automatically deleted
  • Session Data: Retained until you clear session or uninstall the App
  • Chatbot Messages: Last 50 messages persisted for conversation continuity
  • Terminal Recordings: Retained until you manually delete them or uninstall
  • Custom Themes: Retained until you delete the theme or uninstall
  • Connection History: Up to 10 recent entries; older entries automatically removed
  • Device Identifier: Retained until you uninstall the App

Your Data Rights

Access Your Data

You can view your stored audit logs within the App's settings.

Export Your Data

Audit logs can be exported as JSON format from the App settings.

Delete Your Data

  • Uninstalling the App removes all locally stored data
  • You can clear specific data (saved connections, audit logs) from App settings

You have the right to access, rectify, erase, restrict processing, data portability, and object to processing of your personal data. To exercise these rights, use the in-app data management features or contact us.

For California Residents (CCPA/CPRA)

You have the right to know what personal information is collected, delete your personal information, opt-out of the "sale" of personal information (we do not sell data), and non-discrimination for exercising your rights.

Third-Party Services

  • Anthropic, PBC — AI chatbot responses powered by Anthropic's Claude API. See Anthropic's Privacy Policy
  • Fly.io — Proxy server infrastructure. The proxy does not store message content

The App is not intended for children under 13 years of age. We do not knowingly collect personal information from children.

Data Processing Location

Most data processing occurs locally on your device. When using the AI chatbot in free/pro mode, message data is processed through our proxy server (Fly.io, US-East region) and Anthropic's API servers. In BYOK mode, message data is sent directly to Anthropic's servers.

TerminaLLM is designed with privacy as a core principle — we collect minimal data, store credentials locally, and only transmit data to external services when required for specific features.
